# Creonex.viz - Portfolio Website

A stunning, modern portfolio website for Creonex.viz showcasing outfit visual design services.

## 🎨 Features

- **Premium Dark Theme** with vibrant gradients and glassmorphism effects
- **Smooth Animations** and micro-interactions throughout
- **Fully Responsive** design that works beautifully on all devices
- **SEO Optimized** with proper meta tags and semantic HTML
- **Accessibility Features** including keyboard navigation
- **Modern Typography** using Google Fonts (Outfit & Playfair Display)

## 🚀 Quick Start

The website is currently running on your local server!

**Access it at:** http://localhost:8080

The server is already running in your terminal. Just open the URL above in your browser.

## 📝 Customization

To customize the content:
1. Edit `index.html` for text and structure
2. Modify `styles.css` for colors and styling
3. Update `script.js` for interactive features

### Quick Color Changes
All colors are defined as CSS variables in `styles.css`:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --secondary-gradient: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
    /* ... more variables */
}
```

## 🚀 Deployment

To deploy this website:

### Option 1: GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Select main branch as source

### Option 2: Netlify
1. Drag and drop the folder to Netlify
2. Or connect your GitHub repository

### Option 3: Vercel
1. Import the project from GitHub
2. Or use Vercel CLI: `vercel`

### Option 4: Traditional Hosting
Upload all files to your web hosting via FTP

## 🎨 Design Philosophy

This website follows modern web design principles:
- **Visual Excellence** - Premium aesthetics that wow users
- **User Experience** - Smooth interactions and clear navigation
- **Performance** - Optimized animations and lazy loading
- **Accessibility** - Keyboard navigation and semantic HTML
- **Responsiveness** - Perfect on all screen sizes

## 📊 Performance Tips

- All animations use CSS transforms for GPU acceleration
- Intersection Observer for efficient scroll animations
- Debounced scroll events to reduce CPU usage
- Preloaded critical fonts
