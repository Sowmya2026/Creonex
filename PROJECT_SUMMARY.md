## 🚀 **How to Use**

### **Local Development**
```bash
# Navigate to project
cd f:\creonex.viz

# Start server
python -m http.server 8080

# Open browser
http://localhost:8080
```

### **Theme Toggle**
- Click sun/moon icon in top-right
- Theme preference saved automatically
- Logo switches with theme

### **First Visit**
- Intro animation plays once
- Saved in localStorage
- Clear cache to see again
